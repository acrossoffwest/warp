# Session Memory Board Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Warp's local Session Memory Board: interrupted terminal recovery, Claude Code/Codex chat indexing, and safe restore into the correct Warp tab/pane and working directory.

**Architecture:** Add a local `session_memory_records` registry backed by SQLite, expose it through a focused `SessionMemoryModel`, and render it through a dedicated board pane opened from startup recovery and the command palette. Existing `feat/remote-control` agent resume helpers remain the source of truth for Claude/Codex commands, with saved launch flags taking precedence for restored sessions.

**Tech Stack:** Rust, Warp app workspace, Diesel SQLite migrations, Warp model events, existing `workspace/view.rs` tab/pane launch helpers, existing `agent_session_reader`, existing `CLIAgent` command helpers.

```rust
impl SessionMemoryStatus {
    pub fn classify_startup(self, closed_intentionally_at: Option<i64>) -> Self {
        match (self, closed_intentionally_at) {
            (SessionMemoryStatus::Live, None) => SessionMemoryStatus::Interrupted,
            (SessionMemoryStatus::Live, Some(_)) => SessionMemoryStatus::UserClosed,
            (status, _) => status,
        }
    }
}

impl SessionMemoryRecord {
    pub fn is_interrupted(&self) -> bool {
        self.status == SessionMemoryStatus::Interrupted
    }

    pub fn matches_query(&self, query: &str) -> bool {
        let query = query.trim().to_lowercase();
        if query.is_empty() {
            return true;
        }
        [
            Some(self.title.as_str()),
            self.summary.as_deref(),
            self.cwd.as_ref().and_then(|p| p.to_str()),
            self.project.as_deref(),
            self.last_command.as_deref(),
            self.native_session_id.as_deref(),
        ]
        .into_iter()
        .flatten()
        .any(|value| value.to_lowercase().contains(&query))
    }
}
```

- [ ] **Step 3: Define model**

`app/src/session_memory/model.rs`:

```rust
use std::sync::mpsc::SyncSender;

use crate::persistence::ModelEvent;
use super::types::{SessionMemoryRecord, SessionMemoryStatus};

pub struct SessionMemoryModel {
    records: Vec<SessionMemoryRecord>,
    model_event_sender: Option<SyncSender<ModelEvent>>,
}

impl SessionMemoryModel {
    pub fn new(
        mut records: Vec<SessionMemoryRecord>,
        model_event_sender: Option<SyncSender<ModelEvent>>,
    ) -> Self {
        for record in &mut records {
            record.status = record
                .status
                .classify_startup(record.closed_intentionally_at);
        }
        Self {
            records,
            model_event_sender,
        }
    }

    pub fn records(&self) -> &[SessionMemoryRecord] {
        &self.records
    }

    pub fn interrupted_records(&self) -> Vec<SessionMemoryRecord> {
        self.records
            .iter()
            .filter(|record| record.status == SessionMemoryStatus::Interrupted)
            .cloned()
            .collect()
    }

    pub fn filtered_records(&self, query: &str) -> Vec<SessionMemoryRecord> {
        self.records
            .iter()
            .filter(|record| record.matches_query(query))
            .cloned()
            .collect()
    }

    pub fn upsert(&mut self, record: SessionMemoryRecord) {
        if let Some(existing) = self.records.iter_mut().find(|r| r.id == record.id) {
            *existing = record.clone();
        } else {
            self.records.push(record.clone());
        }
        if let Some(sender) = &self.model_event_sender {
            let _ = sender.send(ModelEvent::UpsertSessionMemoryRecord { record });
        }
    }

    pub fn delete(&mut self, id: &str) {
        self.records.retain(|record| record.id != id);
        if let Some(sender) = &self.model_event_sender {
            let _ = sender.send(ModelEvent::DeleteSessionMemoryRecord {
                id: id.to_string(),
            });
        }
    }
}
```
